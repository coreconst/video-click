Welcome to Video-Click! Follow these simple steps to download and manage your videos efficiently.

Step-by-Step Guide

1. Install Required Tools

Ensure you have the necessary tools installed on your system, such as Homebrew and ffmpeg. If you haven't installed them yet, follow these instructions:

Homebrew:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

ffmpeg:

brew install ffmpeg

After installation, it's recommended to restart the app.

2. Add Video Links

In the Safari browser popup, click the Select button. Then, hover the cursor over the desired video, and a Save button will appear in the top right corner. After saving, the button text should change to Saved. We can then return to the app.

3. Set Download Path

Use the Browse button to select the directory where you want to save the downloaded videos. The selected path will be used for all future downloads.

4. Download Videos

After adding the links and setting the download path, click the Synchronize button and download icon. Monitor the download progress directly within the app. The current file size will be updated every few seconds.

Tips

Make sure your internet connection is stable for uninterrupted downloads.

Check the download path to ensure there's enough space for the videos.
